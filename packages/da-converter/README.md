```
da-converter convert [infile] [-o] [-f] [-p]

Extend input samples with geo info.

Positionals:
  infile  Input JSON file with IPs

Опции:
  --version       Show version number
  --help          Show help
  -o, --outfile   Write output to given file instead of stdout
  -f, --filter    Skip unresolved IPs
  -g, --geonames  Resolve geonames
  -p, --pretty    Format output JSON
  ```
